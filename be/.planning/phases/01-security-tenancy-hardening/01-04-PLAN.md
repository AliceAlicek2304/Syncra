  <behavior>
    - When a request with valid Tenant headers arrives:
        1. Check Redis for key `tenant_cache:{userId}:{workspaceId}`.
        2. If hit: use cached boolean to allow/deny access.
        3. If miss: query DB, cache the result for 1 hour, and allow/deny.
  </behavior>

    - Try to get value from cache.
    - If miss, perform DB check, then `await cache.SetStringAsync(key, isMember.ToString(), options)`.
    - Use 1 hour sliding expiration or absolute expiration as per D-02.
  </action>
