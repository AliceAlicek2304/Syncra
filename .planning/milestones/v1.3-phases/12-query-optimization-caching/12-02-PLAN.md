# Plan 12-02: WorkspaceAnalyticsService Refactor

## Objective
Reduce memory overhead and DB transfer size by using projections (`.Select()`) instead of loading full entities with `Media` inclusions.

## must_haves
- [ ] `WorkspaceAnalyticsService` uses `IPostRepository.GetAnalyticsDataAsync` or similar projected method.
- [ ] Analytics queries use `.AsNoTracking()`.
- [ ] `Include(Media)` removed from analytics-specific fetch paths.

## Tasks

### Task 12-02-1: Add Projection Method to IPostRepository
Define a method that returns only necessary fields for analytics.

### Task 12-02-2: Implement Projection in PostRepository
Use `.Select(p => new { ... })` and return a lean DTO or tuple.

### Task 12-02-3: Refactor WorkspaceAnalyticsService
Update `GetSummaryAsync` and `GetHeatmapAsync` to use the new repository methods.
