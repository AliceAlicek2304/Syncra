# Plan 12-01: EF Core Migrations for Indexes

## Objective
Accelerate analytics and multi-tenant queries by creating composite indexes on `Posts` and `AuditLogs` tables.

## must_haves
- [ ] Composite index on `Posts(WorkspaceId, Status, PublishedAtUtc)` created.
- [ ] Index on `AuditLogs(WorkspaceId, CreatedAtUtc)` created.
- [ ] Migration generated and applied to local DB.

## Tasks

### Task 12-01-1: Add Index Configurations to AppDbContext
Modify `OnModelCreating` or separate configuration classes to define indexes.

### Task 12-01-2: Generate EF Core Migration
Run `dotnet ef migrations add AddAnalyticsIndexes --project be/src/Syncra.Infrastructure --startup-project be/src/Syncra.Api`.

### Task 12-01-3: Apply Migration
Run `dotnet ef database update --project be/src/Syncra.Infrastructure --startup-project be/src/Syncra.Api`.
