    **Implementation of `ExportCsvAsync`:**

    1. **Calculate `days` from date range:**
       ```csharp
       var days = (int)Math.Ceiling((endUtc - startUtc).TotalDays);
       if (days < 1) days = 1;
       ```

    2. **Fetch data in parallel** (Task.WhenAll):
       - `_analyticsService.GetSummaryAsync(workspaceId, days, ct)` → `Result<WorkspaceAnalyticsSummaryDto>`
       - `_analyticsService.GetHeatmapAsync(workspaceId, days, ct)` → `Result<HeatmapDto>`
       - `_postRepository.GetPublishedPostsForExportAsync(workspaceId, startUtc, endUtc, ct)` → `IReadOnlyList<PostExportDto>`

    3. **Handle failures:** If summary or heatmap fails, log error and return `Result.Failure<byte[]>(error)`.

    4. **Build CSV string** with these sections:

       ```
       # Analytics Export - Syncra.NET
       # Generated: {UtcNow:yyyy-MM-dd HH:mm:ss} UTC
       # Date Range: {startUtc:yyyy-MM-dd} to {endUtc:yyyy-MM-dd}
       # Workspace: {workspaceId}
       
       
       === SUMMARY ===
       Metric,Value
       Total Reach,{summary.TotalReach}
       Engagement Rate (%),{summary.EngagementRate}
       Follower Growth,{summary.FollowerGrowth}
       Total Posts,{summary.TotalPosts}
       
       
       === HEATMAP ===
       Day Of Week,Hour,Score
       {slot.DayOfWeek},{slot.Hour},{slot.Score}
       ...
       
       
       === POSTS ===
       ID,Title Preview,Published At (UTC),Platform,Status
       {post.Id},{escaped TitlePreview},{post.PublishedAtUtc:yyyy-MM-dd HH:mm:ss},{post.Platform},{post.Status}
       ...
       ```

    5. **CSV escaping rules** (MUST implement, per D-01 "opens directly in Excel"):
       - If a field contains `,`, `"`, or newline → wrap the entire field in double quotes
       - If a field contains `"` → escape as `""` (double-double)
       - Apply to: TitlePreview, ContentPreview (if included)
       - Implement as a private helper: `CsvEscape(string? value)`

    6. **Return:**
       ```csharp
       var bytes = Encoding.UTF8.GetBytes(csv);
       return Result.Success(bytes);
       ```

    7. **WeeklyReach data from summary:** Include weekly reach breakdown after summary metrics:
       ```
       Weekly Reach
       Week Start,Reach
       {week.WeekStart},{week.Reach}
       ...
       ```
